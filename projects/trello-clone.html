Trello Clone

A trello clone web application for managing projects.

Demo account: demo@gmail.com password 123123

Technologies

The front-end was created in Vue.js with the help of the UI framework Quasar, and Vuex for state management.


For back-end I used Google's Firebase service.


The application is hosted on netlify.

Functionalities

The application has the main functionalities of Trello.


On the main page users can add, update, and delete boards.


Every board has lists, and every list has cards. Users can reorder lists and cards, and can move cards from one list to another.
Every card can have a description, a color, and a checklist with items inside.
