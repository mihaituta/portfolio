Quiz

Quiz is a project made for college.

VIDEO

Technologies

The application was written in Java using the Jpheonix library and the SceneBuilder tool.

Functionalities

Quiz is a simple application for solving quizzes.


  • The user logs in then chooses a domain and a quiz corresponding to that domain.
  • Each quiz contains 10 questions, the questions are read from a text file inside the application's folder.
  • When a quiz is finished the user can review the results of the solved quiz or choose another quiz to solve.
