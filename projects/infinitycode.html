InfinityCode

InfinityCode is a project made for college.

VIDEO

Technologies

It was created using HTML, CSS, JavaScript, PHP and MySQL.

Functionalities

The website is an imaginary programming contest called InfinityCode.


There are 8 pages: home, rules, sponsors, news, participants, results and contact that can be accessed by anyone, and an administration page that is hidden and protected with name and password for admins.


Users can register to the contest on the participants page.


Administrators can:

  • Add, modify or delete participants.
  • Add, modify, or delete news articles.
